# EasyClip
## 功能
- [x] 剪切板操作实时监控
- [x] 剪切板文字内容实时记录
- [x] 剪切板图片内容实时记录
- [x] 剪切板文件内容实时记录
- [x] 剪切板历史快速搜索
- [x] 双击条目复制
- [x] 剪切板音效
- [x] 开机自启动
- [x] 快捷键调出剪切板历史记录面板/自定义快捷键
- [x] 历史记录面板有左/右侧悬浮、类型QQ鼠标接触左/右上角动态显示、浮动与所有窗体之前等多种模式
